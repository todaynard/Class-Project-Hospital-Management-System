# 🏥 Hospital Management System

## 📌 Project Description
This project is a Java-based Hospital Management System developed using Object-Oriented Programming (OOP) principles.

It models real-world hospital operations such as managing doctors, patients, and appointments.

The system demonstrates:
- Abstraction (abstract classes)
- Inheritance (IS-A relationship)
- Interfaces
- Encapsulation (private variables with getters)
- Method overriding
- Constructor usage

Additionally, the system allows dynamic user input and handles errors using exception handling.

---

## ⚙️ Features
- Add doctor details
- Add patient details
- Schedule appointments
- Calculate patient bills
- Accept user input at runtime
- Prevent program crashes using exception handling

---

## ⚠️ Exceptions Implemented

### 1. InputMismatchException
- Occurs when the user enters the wrong data type
- Example: entering text instead of a number

### 2. IllegalArgumentException
- Used when invalid values are entered
- Examples:
  - Negative age
  - Empty name

### 3. General Exception
- Catches unexpected errors to prevent program crash

### 4. finally Block
- Ensures resources like Scanner are closed properly

---

## ▶️ How to Run the Program

1. Compile all files:
   javac *.java

2. Run the program:
   java Main

3. Follow the prompts and enter required details.
