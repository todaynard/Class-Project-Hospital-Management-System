# Hospital Management System

## Project Description
This is a Java OOP project that simulates a simple hospital management system using concepts like inheritance, abstraction, and interfaces.

## Features
- Manage doctors and patients
- Schedule appointments
- Calculate patient bills

## How to Run
1. Compile all files:
   javac *.java

2. Run:
   java Main
